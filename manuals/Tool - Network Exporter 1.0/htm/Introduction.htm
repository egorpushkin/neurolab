Network Exporter Tool 1.0 - User Guide
Overview
Introduction
Network Exporter Tool was created to automatize exporting Neuro Laboratory projects with all necessary materials.
Execution
After normal application installation this tool can be started by executng file NetworkExporter.exe.
Network Exporter Features
You can perform the following using Network Exporter Tool:
  • Export neural network project with libraries and Network Importing SDK source code for latest usage in your custom applications.
  • You have an opportunity to export currently loaded network project if the tool is used in cooperation with Neuro Laboratory Environment.
