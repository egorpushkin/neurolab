Network Exporter Tool 1.0 - User Guide
User Interface Quick Tour
The main idea of Network Exporter Tool is exporting Neuro Laboratory project, all necessary libraries and source code for latest usage in your custom applications. It can be done in three easy steps:
  1. You must select project file to export.
  2. You must select destination folder.
  3. You must follow Export... button located at the main application window.
If you are running Network Exporter Tool from Neuro Laboratory Environment Current Neuro Laboratory project radio button will be active. Before executing this tool Neuro Laboratory Environment stores currently opened project in the temporary file which would be exported if you choose this variant. You can see where this temporaty file has been stored using Details... button. The only item at the Project Files sections specifies project file to export.
Otherwise (if you select Custom project radio) you will have to specify project file name to export with the solution. If you are going to export currently opened project and specify it's location be sure that you have saved made changes otherwise you would export old project file.
