Toolbox - Network Elements
Introduction
All elements divided in several groups:
  • Data Flow Elements
  • Data Processors
  • Layers
Data Flow Elements Group
This group has the following elements:
  • Input
  • Input Factory
  • Constant Input
  • Output
  • Output Factory
This group contains elements for distributing input data between processing elements of the first layer and amalgamation of output signals from the final neurons layer.

Layers Group
This group has the following elements:
  • Hopfield Layer
  • Neurons Layer
  • RBF Layer
  • Winner Layer
Elements of this group helps to build network for processing large dimension data using minimal number of blocks.
