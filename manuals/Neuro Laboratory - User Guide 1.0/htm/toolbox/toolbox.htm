Toolbox
Neuro Laboratory provides you an opportunity to build, configure and train neural network. There ase some toolboxes that contains required elements and additional tools for building network of required structure.
Network Elements Toolbox
This toolbox contains blocks for building network structure.
Neuron Functions Toolbox
The theory of neural network is built so that every neuron calculates it's output (value that can be accessed from it's axon) manipulating with data taken from it's inputs (dendrites). After output is calculated it is often being passed through transfer or limiter function. This toolbox contains several different functions that can be applied to netwirk elements.
