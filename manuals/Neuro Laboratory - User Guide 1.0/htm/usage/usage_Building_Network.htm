Building Network
Introduction
This sample ilustrates process of building network using Neuro Laboratory Environment. It is reqired to build network of the following structure:

Figure 1. Network Logic Structure.
We are going to realize it using network model of the following structure (in Neuro Laboratory terminology):

Figure 2. Network Elements Structure.
Building Network
To build such a network you must follow these steps:
  1. Add three Input Factory elements. Configure them in the following way:
    Input Factory - IF 1:
      Inputs count: 16.
      Layer dimension: 16.
    Input Factory - IF 2:
      Inputs count: 64.
      Layer dimension: 64.
    Input Factory - IF 3:
      Inputs count: 32.
      Layer dimension: 32.
Note To configure any layer element you must click right mouse button over it and select Properties... menu item.
Note It is requied to set identical values for Inputs count and Layer dimension settings of input and output layer elements (Input Factory and Output Factory).
  1. Add RBF Layer element and configure it in the following way:
    RBF Layer - RBFL 1:
      Inputs count: 48.
      Layer dimension: 16.
      Activation function: Exponent.
Activation function can be specified using Element Properties control panel. Select element you wan to configure and press Select button on the panel.

Figure 3. Elements Properties Control Panel.
  1. Add Neurons Layer element and configure it in the following way:
    Neurons Layer - NL 1:
      Inputs count: 65.
      Layer dimension: 16.
      Activation function: Sigmoid.
  2. Add Constant Input element and configure it in the following way:
    Constant Input - C 1:
      Value: -1.
  3. Add Winner Layer element and configure it in the following way:
    Winner Layer - WL 1:
      Inputs count: 32.
      Layer dimension: 32.
  4. Add Neuron element and set it's activation function to Linear.
  5. Add Output element.
  6. Add Output Factory element and configure it in the following way:
    Output Factory - OF 1:
      Inputs count: 30.
      Layer dimension: 30.
After all elements has been added we have to establish connections between them in required way.
  1. Establish connections between the following elements:
    IF 1 - RBFL 1:
      Connections count: 16.
    IF 2 - RBFL 1:
      Connections count: 32.
    IF 2 - NL 1:
      Connections count: 32.
    IF 3 - NL 1:
      Connections count: 32.
    C 1 - NL 1:
      Connections count: 1.
    RBFL 1 - WL 1:
      Connections count: 16.
    NL 1 - WL 1:
      Connections count: 16.
    WL 1 - N 1:
      Connections count: 2.
    WL 1 - OF 1:
      Connections count: 30.
You can configure connection settings using Element Properties control panel. You must select element and then select connection that must be configured (from Dendrites or Axons window).
Atfer network has been built you can verify it using Network Verification control panel. Prepared network must looks like this:

Figure 4. Prepared Network View.
You can find prepared project file at Samples folder. It is called NetworkProject.nlproj.
