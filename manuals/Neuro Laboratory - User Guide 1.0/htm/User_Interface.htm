Neuro Laboratory 1.0 - User Guide
User Interface Quick Tour
Introduction
Neuro Laboratory Environment is the main part of Neuro Laboratory 1.0 solution.
Main Application Window
Neuro Laboratory Environment main window has the following sections:
  • Builder Workspace - Performs functionality for building neural network.
  • Network Elements Toolbox - Contains a list of available network elements.
  • Plug-ins Panel - Contains a list of loaded plug-ins.
  • Control Panels Section

Figure 1. Main Neuro Laboratory Window.
Builder Workspace Section
This section presents functionality for building and configuring neural network. You must use Network Elements Toolbox section to add elements to project (simply drag one of those elements from toolbox area to the Builder Workspace to perform it).
After element has been added you can remove it or connect to other network elements from your project. You can access element's control menu by clicking on it using right mouse button.

Figure 2. Elements' Control Menus.
Use Add Dendrite/Add Layer Input to specify data source element. Use Add Axon/Add Layer Output to specify data destination element. You can configure connection settings using Element Properties control panel (it is often required to specify connection length after establishing connection). It is also available to specify element's activation function using this panel (by default it is set to Single Change).
Note that you cannot add dendrite (input connection) to output element as well as axon (output connection) to input element.
You can also configure layer element using Properties... menu item.

Figure 3. Layer Properties Dialog.
Layer is been created with the following settings by default:
  Inputs count: 64.
  Layer dimension: 64.
All layer weights are initialized to 0.0 (Layer Properties Dialog does not present functionality for manipulating layer weights).
Main Menu Section
Use menu Tools to get access to Neuro Laboratory tools. Data Creator Tool and Network Exporter Tool can be executed this way. You will have an opportunity to export currently opened network project (with made modifications) using Network Exporter Tool running it using corresponding item from this menu.

Figure 4. Menu Tools.
Note that any of these tools would be executed only if it was properly installed. Otherwise you would be notified about the problem.
Control Panels Section
Neuro Laboratory has the following control panels:
  • Element Properties
  • Neuron Functions
  • Training Network
  • Network Verification
  • Plugin Bar
Neuron Functions panel is used to display a list of all available activation functions. You can also configure function's arguments and see it's graph.

Figure 5. Panel Neuron Functions.
Training Network panel is used to train built network using one of external trainers. You must select trainer and load data to process it.

Figure 6. Panel Training Network.
After data file is loaded and trainer is selected press Start Training button to show Trainer Tool. You can configure trainer at any time by pressing Configure button (located at Neuron Functions panel) or Configure Trainer button (located at Trainer Tool dialog).

Figure 7. Trainer Tool Dialog.
You can start training using Start button.
Network Verification panel is used to perform network verification. This operation includes the following tests:
  • Check Elements Presence.
    Possible notifications:
      Warning 0001 - Network does not contain any elements.
  • Check Network Content.
    Possible notifications:
      Error 0012 - Network does not contain any input elements.
      Error 0013 - Network does not contain any output elements.
  • Check For Single Elements.
    Possible notifications:
      Error 0001 - Element '{Element Name} {Element ID}' has no any connections.
  • Check For Wrong Processors.
    Possible notifications:
      Error 0002 - ET_PROCESSOR element '{Element Name} {Element ID}' has no any input connections.
      Error 0003 - ET_PROCESSOR element '{Element Name} {Element ID}' has no any output connections.
  • Check Layer Elements.
    Possible notifications:
      Error 0004 - ET_INPUT element '{Element Name} {Element ID}' is connected incorrectly.
      Error 0005 - ET_OUTPUT element '{Element Name} {Element ID}' is connected incorrectly.
      Error 0006 - ET_PROCESSOR element '{Element Name} {Element ID}' has wrong input connections.
      Error 0007 - ET_PROCESSOR element '{Element Name} {Element ID}' has wrong output connections.
      Error 0008 - Element '{Element Name} {Element ID}' must have the same dimension and inputs count.
  • Data Flow Test.
    Possible notifications:
      Error 0010 - Data flow test skipped. Network contains structure errors.
      Error 0009 - Prepared network contains at least one circle.
      Error 0011 - Network circle found:
         (list of elements which make up a cicle would be displayed below).
It is highly recommended to perform verification and correcting found errors before processing any operations over the network such as data flow, executing trainers or plug-ins.

Figure 8. Panel Network Verification.
