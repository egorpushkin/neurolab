Network Importing SDK Documentation
Introduction
One of the main Neuro Laboratory concepts is usage prepared materials for customer's purposes. You have an opportunity to create application which will use Neuro Laboratory project files. So you can build and train any neural network you want using the environment and then exoprt it for latest usage in your custom solution. Network Importing SDK provides you all necessary documentation and source code that required for connecting SDK binary libraries and operating with network object model built in Neuro Laboratory.
The department also includes basics of the object model which was built for Neuro Laboratory.
Reference deparment contains the enumeration and complete description of classes and interfaces which are used for manipulating with network object model.
Network Importing SDK Features
  • Contains libraries that implements functionality of network serialization and processing mechanisms.
  • Contains source code for integrating binary libraries with your applications.
  • Contains complete documentaion for developers.
Network Importing SDK Source
This SDK release includes source code required for using binary libraries in your applications. After normal installation you can find these materials in the following folder:
"$Solution_Dir"\SDK\Network Importing\Source
This folder contains Microsoft Visual Studio 7.1 project NetworkImportingCommon and CImportManagerComponent header and implementation files (ImportManagerComponent.h and ImportManagerComponent.cpp).
Note This code refers to Data Processing SDK source so be attentive if you are going to move this project to another location. In this case you will have to update link in file NetworkImporting.h.
You must use Network Exporter Tool to export Neuro Laboratory project files for latest usage in your custom appliactions. During exporting project file this tool also exports all necessary source code (Network Inporting as well as Data Processing). See Appendix A - Exported Solution Structure to get more information about exported materials structure. It is recommended to use source located at the folder that you will specify during exporting.
Perform the following to add functionality of Network Importing SDK to your application:
  1. Add NetworkImportingCommon and DataProcessingCommon projects to your solution.
  2. Add ImportManagerComponent.h and ImportManagerComponent.cpp to one of your projects.
  3. Include ImportManagerComponent.h and declare object of CImportManagerComponent class.
  4. Execute CImportManagerComponent::LoadComponent method to initialize component.
Note Do not forget to execute CImportManagerComponent::UnloadComponent to unload component and release all allocations.
Note This source uses functionality of Microsoft Foundation Class (MFC) Library. It must be included in MFC project (MFC Application or MFC DLL).
