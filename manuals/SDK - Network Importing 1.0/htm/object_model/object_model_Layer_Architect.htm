Network Layer Internal Data Model
Introduction
Layer Controls are the most often used network elements. Every layer element can collect as many elementary elements of the same type as necessary. For example Neurols Layer element can contain from 1 to 65535 single neurons. A number af layer input connections is also limited to 65535.
The following network elements supports Layer Control architecture:
  • Input Factory
  • Output Factory
  • Hopfield Layer
  • Neurons Layer
  • RBF Layer
  • Winner Layer
Layer Component View
Layer Control contains three independent parts:
  • Input Data Factory. This factory object is used to store input data collected from layer input connections.
  • Weights Matrix. Weights matrix presents an array of CDataFactory objects that stores layer weights. Every data factory object corresponds to layer input connection (so amount of factories equals to amount of layer input connections). The length of data factory from this array equals to layer's dimension. Figure 3 illustrates data model that created in every layer element. Weights matrix also contains layer Deltas factory. This factory was added to store additional information about elementary layer elements. Every element can store one float value in the corresponding cell of this factory. So the dimension of this factory equals to the dimension of layer element.
  • Output Data Factory. This factory object is used to store layer output data during layer processing before performing data flow through this element.
