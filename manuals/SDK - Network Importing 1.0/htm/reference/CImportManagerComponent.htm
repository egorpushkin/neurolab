CImportManagerComponent Class 
class CImportManagerComponent : public IComponent
Remarks
This class contains code for loading NetworkImport.dll library. You must include this code into your project.

CImportManagerComponent members
The only CImportManagerComponent method should be mentioned is GetImportManager. This method returns IImportManager object. Return value differs form NULL only if CImportManagerComponent object has been properly initialized.
