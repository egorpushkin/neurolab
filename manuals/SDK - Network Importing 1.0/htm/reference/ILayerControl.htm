ILayerControl Interface 
interface ILayerControl : public IObject
Remarks
Performs manipulation mechanism for network layer object. See Network Layer Internal Data Model to get more information about layer objewcts internal structure.

ILayerControl members
Common operations
SetInputData Sets layer input data.
SetOutputData Sets layer output data.
ModifyWeights Executes internal weights modification mechanism.
ResetWeights Changes layer weights to the specified value.
CreateWeights Resizes layer's weights factory. All previos weights' values are being lost after this call.
RemoveWeights Releases layer's weights factory.
GetDimension Returns layer dimension.
GetInputsCount Returns layer's inputs count.
GetWeight Returns layer's weight at the specified position.
SetWeight Updates layer's weight at the specified position.
SetDelta Returns layer's delta at the specified position.
GetDelta Updates layer's delta at the specified position.

// ILayerControl declaration
interface ILayerControl : public IObject
{
    virtual void		SetInputData(IDataFactory* pData) = 0;
    virtual void		SetOutputData(IDataFactory* pData) = 0;

    virtual bool		ModifyWeights() = 0;
    virtual void		ResetWeights(float Value) = 0;

    virtual void		CreateWeights(int Inputs, int Dimension) = 0;
    virtual void		RemoveWeights() = 0;

    virtual int		GetDimension() = 0;
    virtual int		GetInputsCount() = 0;

    virtual float		GetWeight(int i, int j) = 0;
    virtual void		SetWeight(int i, int j, float fWeight) = 0;

    virtual void		SetDelta(int i, float Value) = 0;
    virtual float		GetDelta(int i) = 0;
};
