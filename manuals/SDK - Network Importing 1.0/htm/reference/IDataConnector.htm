IDataConnector Interface 
interface IDataConnector : public IObject
Remarks
The IDataConnector interface performs functionality for operating with element's input and output connections.
Supported Interfaces
Interface Name Interface Identificator Declared In
IElement IDD_IElement CmnElement.h
IDataFlow IDD_IDataFlow CmnElement.h
ILayerControl IDD_ILayerControl CmnElement.h
Note The ILayerControl interface supported by the following elements only: Input Factory, Output Factory, Hopfield Layer, Neurons Layer, RBF Layer, Winner Layer
IDataConnector members
Public Methods
GetConnectionsFactory Returns connection factory of the specified type.
Internal Methods
AddConnectionTo Adds new connection to the specified element.
RemoveConnection Removes specified connection.
SetAddConnectionState Performs configuration of the internal connections manager.
ProcessAddConnection Adds new connection to the specified element.
MoveConnection Moves connection in connections list.
FindConnectionByPrevPtr Returns connection object by the specified previous pointer value.
Note As you can see all methods of this interface are public (in C++ terminology). IDataConnector methods form the second group (Internal Methods) presentes functionality for internal mechanisms. It is not recommended to execute any of these methods.
Source
Source code that declares IDataConnector interface: 
// IDataConnector declaration
interface IDataConnector : public IObject
{
    virtual void		AddConnectionTo(IElement* pElement, int FactoryType) = 0;
    virtual void		RemoveConnection(CConnection* pConnection) = 0;

    virtual void		SetAddConnectionState(int iState, int Count) = 0;
    virtual void		ProcessAddConnection(IElement* pElement) = 0;

    virtual bool		MoveConnection(CConnection* pConnection, bool bIsMoveUp = false) = 0;

    virtual IConnectionsFactory* GetConnectionsFactory(int FactoryType) = 0;

    virtual CConnection*	FindConnectionByPrevPtr(CConnection* pConnection) = 0;
};
