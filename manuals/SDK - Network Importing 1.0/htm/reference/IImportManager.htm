IImportManager Interface 
interface IImportManager : public IObject
Remarks
Library NetworkImport.dll implements main serialization and network processing mechanisms. It contains object that implements IImportManager. This control interface provides client application an opportunity to manage library internal state and manipulate with neural network object model.
The object that inplements IImportManager also provides access to other network control interfaces. Any of that interfaces can be queried using IImportManager::QueryInterface call.
