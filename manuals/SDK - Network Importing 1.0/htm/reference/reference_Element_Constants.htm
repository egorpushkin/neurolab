Network Importing SDK Documentation
Element Constants
The following constants specify type of network element. You can receive element's type using IElement::GetElementType call.
Constants declaration: 
// Network element types
#define ET_INPUT				0x000003
#define ET_OUTPU				0x000004
#define ET_PROCESSOR			0x000005
#define ET_BENCHMARK			0x000006
The following constants specify type of connections factory.
Constants declaration: 
// Connections factory types
#define DC_INPUTS_FACTORY			0x000001
#define DC_OUTPUTS_FACTORY			0x000002
Requirements
Header: Declared in: DefElement.h, include NetworkImporting.h
