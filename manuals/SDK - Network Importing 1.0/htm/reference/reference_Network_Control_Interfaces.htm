Network Importing SDK Documentation
Network Control Interfaces
Introduction
These interfaces provide you full control over neural network.
IElementsCollection Overview
This interface provides enumeration mechanism. You can parse network structure and perform required manipulations.
Source
IElementsCollection declaration: 
interface IElementsCollection : public IObject
{
    virtual int		GetInputsCount() = 0;
    virtual int		GetOutputsCount() = 0;

    virtual void		CreateElementsEnumerator(int EnumeratorIndex) = 0;
    virtual IElement*	GetNextElement(int EnumeratorIndex) = 0;
};
INetworkProcessor Overview
This interface provides you access to network global data flow mechanism. ProcessNetwok function performs data flow through the network.
Source
INetworkProcessor declaration: 
interface INetworkProcessor : public IObject
{
    virtual void		ProcessNetwork() = 0;

    virtual void		SetInputData(IDataFactory* pData) = 0;
    virtual void		SetOutputData(IDataFactory* pData) = 0;

    virtual void		GetInputData(IDataFactory* pData) = 0;
    virtual void		GetOutputData(IDataFactory* pData) = 0;
};
Architecture Details
These interfaces are implemented by CNeuroFactory environment object. This object also contains list of all network elements.
