IElement Interface 
interface IElement : public IObject
Remarks
This interface is basic for the network element object model built in Neuro Laboratory. You can get access to other network element's control interfaces using IElement::QueryInterface method.
Supported Interfaces
Interface Name Interface Identificator Declared In
IDataFlow IDD_IDataFlow CmnElement.h
IDataConnector IDD_IDataConnector CmnElement.h
ILayerControl IDD_ILayerControl CmnElement.h
Note The ILayerControl interface supported by the following elements only: Input Factory, Output Factory, Hopfield Layer, Neurons Layer, RBF Layer, Winner Layer
IElement members
Public Methods
GetElementID Returns element's identifier. This identifier is unique for all elements of the same type (for example, for RBF Layer elements).
GetElementTitle Returns element's title (For example, Neurons Layer).
GetElementType Returns element type (ET_INPUT, ET_OUTPUT, ET_PROCESSOR).
GetSysName Returns element's system name (For example, NeuronsLayer).
Internal Methods
CommandNotify Executes element's internal function.
SetElementID Sets element's identifier.
SetSysName Sets element's system name.
SetSysTitle Sets element's title.
Note As you can see all methods of this interface are public (in C++ terminology). IElement methods form the second group (Internal Methods) presentes functionality for internal mechanisms. It is not recommended to execute any of these methods.

interface IElement : public IObject
{

    virtual void		SetElementID(int iID) = 0;
    virtual int		GetElementID() = 0;

    virtual CString	GetElementTitle() = 0;

    virtual int		GetElementType() = 0;

    virtual void		SetSysName(CString& Name) = 0;
    virtual CString	GetSysName() = 0;
    virtual void		SetSysTitle(CString& Title) = 0;
};
