IConnectionsFactory Interface 
interface IConnectionsFactory : public IObject
Remarks
This interface provides mechanism for managing element's connections. Every network element contains two factory objects which implements this interface.
IConnectionsFactory members
Public Methods
GetConnectionsCount Returns connections count.
CreateEnumerator Initialises connections enumeration machine.
GetNextConnection Returns current connection and moves pointer to the next one element. Finally returns NULL.
Internal Methods
RemoveConnections Removes all foctory connections.
AddConnection Add new connection.
RemoveConnection Removes specified connection.
MoveConnection Moves connection in connections list.
Note As you can see all methods of this interface are public (in C++ terminology). IConnectionsFactory methods form the second group (Internal Methods) presentes functionality for internal mechanisms. It is not recommended to execute any of these methods.
Source
Source code that declares IConnectionsFactory interface: 
// IConnectionsFactory declaration
interface IConnectionsFactory : public IObject
{
    virtual void		RemoveConnections() = 0;

    virtual void		AddConnection(CConnection* pConnection) = 0;
    virtual void		RemoveConnection(CConnection* pConnection) = 0;

    virtual bool		MoveConnection(CConnection* pConnection, bool bIsMoveUp = false) = 0;

    virtual int		GetConnectionsCount() = 0;
    INTERFACE_ENUMERATOR_DECLARATION(CConnection*, CreateEnumerator, GetNextConnection)	
};
