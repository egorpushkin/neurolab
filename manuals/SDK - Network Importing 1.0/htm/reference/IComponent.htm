IComponent Interface 
interface IComponent : public IObject
Remarks
Neuro Laboratory components model is based on IComponent and IManagerControl interfaces. This interface has been included in this SDK package because it was decided to derive CImportManagerComponent from this interface.
This interface is often implemented on client's side and contains (in it's implementation) code for loading any external library or connecting external object.
Source

#pragma once

typedef CString	(*PGET_COMPONENT_NAME_ROUTINE)(void);
typedef CString	(*PGET_COMPONENT_VERSION_ROUTINE)(void);

// IComponent declaration
interface IComponent : public IObject
{
public:
    virtual bool		LoadComponent() = 0;
    virtual bool		UnloadComponent() = 0;
    virtual bool		VerifyWhetherExists() = 0;

    virtual CString	GetComponentName() = 0;
    virtual CString	GetComponentVersion() = 0;
};
