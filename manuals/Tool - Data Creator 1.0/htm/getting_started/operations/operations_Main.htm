Data Creator Tool 1.0 - User Guide
Operations
You can perform the following using Data Creator tool:
  • Create New File
  • Load Existing File
  • Save Data File
  • Add Symbol
  • Remove Symbol
  • Edit Symbol
  • Manage Data Image Control Properties
  • Use Advanced Image Editor
Create New File
Push button to create new data file. You would be notified if currently opened data file has been modified since the last save operation.

Figure 1. New File Dialog.
You must specify length of input and output symbols to create new data file.
Load Existing File
Push to load existing data file.
Save Data File
Push to save made changes. You must use button to save data file for the first time.
Add Symbol
You can perform this operation after creating or loading data file. Push corresponding button to add new symbol. Data factory associated with the symbol would be filled with current input marker value.
Remove Symbol
Push corresponding button to remove current data symbol.
Edit Symbol
Use Image Editor to edit currrently displayed data symbol.
Manage Data Image Control Properties

Figure 2. Data Image Control Properties Dialog.
Use Advanced Image Editor
You can load data symbol from external file (*.bmp and *.pgm file types supported). Use File - Open... menu of Advanced Image Editor to process it. You can load only data symbol with the same length as the length of corresponding symbols container.
You can also fill data symbol with specified value using Operations section of Advanced Image Editor main window.

Figure 3. Advanced Image Editor Dialog.
