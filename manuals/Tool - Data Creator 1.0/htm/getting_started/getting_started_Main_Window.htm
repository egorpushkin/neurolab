Data Creator Tool 1.0 - User Guide
Main Application Window
The following sections lcated at the main Data Creator window:
  • Common Information Section
  • Input Symbols Section
  • Output Symbols Section
  • File Global Operations Section
  • System Menu Section

Figure 1. Main Data Creator Tool Window.
Common Information Section
This section displays information about currently opened file name, length for input and output symbols and indicator which shows whether currently opened data file has been modified.
Input Symbols Section
This section allows managing data file imput symbols. The following operation can be performed in this section:
Adds new input symbol to data file. Data factory associated with the symbol would be filled with current input marker value.
Removes current input symbol from data file.
Shows Data Image Control Marker dialog.
Shows Advanced Image Editor dialog.
Shows Data Image Control Properties dialog.
Displays previous data symbol in the Data Image Control.
Displays next data symbol in the Data Image Control.
This Data Image Control allows preparing requierd symbols.
This section also contains the following indicators:
  • Symbol number
  • Total symbols
  • Current marker
Output Symbols Section
Output Symbols section has the same controls as Intput Symbols one and performs the same functionality.
File Global Operations Section
The following operation can be performed using this section:
Shows Creating New Data File dialog.
Loads existing data file.
Saves data file to current location.
Saves data file to new location that must be specified.
System Menu Section

Figure 2. System Menu.
You can access User Guide, Data Processing SKD Documenation and About departments of Data Creator Tool using this menu.
