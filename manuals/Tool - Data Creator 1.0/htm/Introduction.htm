Data Creator Tool 1.0 - User Guide
Overview
Introduction
Data Creator Tool was created for managing data files used for training and processing neural networks in the Neuro Laboratory Environment.
Execution
After normal application installation this tool can be started by executng file DataCreator.exe.
Data Creator Features
You can perform the following using Data Creator Tool:
  • Create/Load data file with various number of input and output symbols.
  • Add/Remove/Modify any of existing data file symbols.
  • Paint any symbol you require using built-in editor.
  • Load symbol from external file (*.bmp and *.pgm file types supported).
You can get mode information about data files format and architecture using Data Processing SDK documentation.
See Getting Started department for more information about Data Creator user interface.
