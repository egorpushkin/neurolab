Source Code
The following source files are included in the DataProcessingCommon project:
  • DataProcessing.h
  • CmnObjects.h
  • CmnStorage.h
  • CmnXML.h
  • CmnDataFactory.h
  • CmnPGMFactory.h
  • CmnBMPFactory.h
  • CmnDataFile.h
  • interface_idds.h
Include DataProcessing.h in your project to add functionality of this SDK.
This source uses functionality of Microsoft Foundation Class (MFC) Library. It must be included in MFC project (MFC Application or MFC DLL).
