Data Processing SDK Documentation
Introduction
Data Processing SDK was created for the better understanding of data model and data processing mechanisms which were used in Neuro Laboratory. Neuro Laboratory Environment can be extended by writing additional modules as well as functionality of the solution can be used in user's applications. To use any of these opportunities you must understand the basic concepts of data flow processes and data maniplations which are implemented in Neuro Laboratory. This SDK documentation imcludes complete data processing reference.
The department also includes basics of the object model which was built for Neuro Laboratory.
Reference deparment contains the enumeration and complete description of classes and interfaces which are used for data processing.
Data Processing SDK Features
  • Includes source code of the main data processing mechanisms used in Neuro Laboratory.
  • Contains complete documentaion for developers.
Data Processing SDK Source
This SDK release also includes source code of main data processing mechanisms that where used in Neuro Laboratory. After normal installation you can find these materials in the following folder:
"$Solution_Dir"\SDK\Data Processing\Source
This folder contains Microsoft Visual Studio 7.1 project DataProcessingCommon.
Perform the following to add functionality of Data Processing SDK to your application:
  1. Add DataProcessingCommon project to your solution.
  2. Include DataProcessing.h file.
Note This source uses functionality of Microsoft Foundation Class (MFC) Library. It must be included in MFC project (MFC Application or MFC DLL).
