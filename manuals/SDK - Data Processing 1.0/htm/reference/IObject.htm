IObject Interface 
interface IObject
Remarks
The IObject interface is basic for the object model built in Neuro Laboratory. All other interfaces implement it. The IObject interface lets clients get pointers to other interfaces on a given object through the QueryInterface method.
IObject members
GetClassString Returns class string for object that implements IObject interface.
QueryInterface Returns pointers to supported interfaces.
IObject::GetClassString 
virtual CString GetClassString() = 0;
IObject::QueryInterface 
virtual void QueryInterface(
    int iIDD, 
    void** ppvObject
) = 0;
IObject::QueryInterface Parameters
iIDD
[in] This parameter specifies interface identificator to be retrieved.
ppvObject
[out] Address of pointer variable that receives the interface pointer requested in iIDD. Upon successful return, *ppvObject contains the requested interface pointer to the object. If the object does not support the interface specified in iIDD, *ppvObject is set to NULL.
Sample
Sample of IObject core usage: 
// pElement - pointer to some element object
IElement* pElement;
ASSERT(pElement != NULL);
	
ILayerControl* pLayerControl = NULL;
pElement->QueryInterface(IDD_ILayerControl, (void**)&pLayerControl);

if (pLayerControl)
{
    // pElement supports ILayerControl
} else
{
    // ILayerControl interface not supported
}
