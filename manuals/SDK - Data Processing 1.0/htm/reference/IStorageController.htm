IStorageController Interface 
interface IStorageController : public IObject
Remarks
The IStorageController interface provedes methods for creating and managing storage objects with hierarchy structure.
Associated identifacator - IDD_IStorageController.
IStorageController members
Serialize Loads or stores an object from/to an archive.
IStorageController::Serialize 
virtual bool Serialize(
    CArchive& ar
) = 0;
IStorageController::Serialize Parameters
ar
[in/out] A CArchive object to serialize to or from.
IStorageController::Serialize Remarks
You must override this function for all classes derived from IStorageController interface.
Use CArchive::IsLoading or CArchive::IsStoring to determine whether the archive is loading or storing.
Note IStorageController interface is a child of IObject so it inherits it's pure methods (GetClassString and QueryInterface) and they must be overridden in class that implements IStorageController.
Sample
Declaring a simple class that implements IStorageController. 
class CSimpleStore :
    public IStorageController
{
public:
    // IStorageController implementations section
    virtual bool Serialize(CArchive& ar)
    {
        if (ar.IsLoading())
        {
            // add loading code here
        } else
        {
            // add storing code here
        }
    }
	
    // IObject implementations section
    virtual CString GetClassString()
    {
        return _T("CSimpleStore");
    }
	
    virtual void QueryInterface(int iIDD, void** ppvObject)
    {
        *ppvObject = NULL;

        switch(iIDD)
        {
        case IDD_IStorageController:
            *ppvObject = (IStorageController*)this;
            break;		 
        }	
    }
};
