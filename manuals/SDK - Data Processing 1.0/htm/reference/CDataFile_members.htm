CDataFile::LoadFile 
virtual bool LoadFile(
    CString& FileName
);
Loads data from a disk file. Function returns true when loading was successfull.
Note This function releases existing data before trying to process loading but after establishing connection to the file which name is passed to the function as parameter. In case of wrong data serialization all previously contained data will be lost.
CDataFile::LoadFile Return Value
Function returns false when connection to the file cannot be established and in case of wrong data serialization.
