Interface Identificators
Identificators are used to query interface implemented by some object's class or supported by one of object's members (this object must implement IObject interface). Use IObject QueryInterface method to get access to necessary interface.
Note Be sure that the object support interface you want to query (otherwise QueryInterface returns NULL).
It is not recommended to use identificators' numeric values cause it can be changed during development process and can differs from version to version.
