IXMLStorageController Interface 
interface IStorageController : public IObject
Remarks
The IXMLStorageController interface provedes an opportunity of storing and loading XML data.
Associated identifacator - IDD_IXMLStorageController.
IXMLStorageController members
Serialize Loads or stores an object from/to an archive.
IXMLStorageController::Serialize 
virtual bool Serialize(
    IXMLDOMDocument* pXMLDom, 
    bool bIsLoad
) = 0;
IXMLStorageController::Serialize Parameters
pXMLDom
[in/out] IXMLDOMDocument object to serialize to or from.
bIsLoad
[in] You must specify here operation that must be performed. You must pass true to load data from pXMLDom object into memory.
