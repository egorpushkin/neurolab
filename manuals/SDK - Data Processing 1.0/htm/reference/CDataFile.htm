CDataFile Class 
class CDataFile :
    public IDataFile,
    public IStorageController
Remarks
The CDataFile class provides functionality of creating and managing two-streamed data file objects.
You can create new empty object by calling CreateNewFile function.
This class implements interface IStorageController so you can get access to it's member function Serialize. Otherwise you can process save/load operation on the object using native IDataFile operators: LoadFile and SaveToFile.
Note All CDataFile members are derived from IDataFile interface.
