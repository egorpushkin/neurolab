Registration
Registration is compalsory procedure which must be performed by any client to download and run any product from Neuro Laboratory 1.0 products family.
You must process the following steps to buy and register any Scientific Software application:
  1. Create personal client accont at http://www.scientific-soft.com/. This is simple one-step procedure. You will have to specify the following: first name, last name, country, city, address, zip/postal code, phone number, e-mail, username and passphrase.
Note You must specify valid information during registration. In other case your personal department can be locked by the site administration.
  1. Add product purchase request for the selected solution at Purchase Department.
  2. Perform required payment. You can get more information about processing payments at Purchase Request page.
  3. After payment would be accepted you will have an opportunity to download solution package.
  4. After downloading and installing your application copy you will have to generate public key and place key file request at the corresponding Purchase Request page. You can create your public key using License Manager tool.
Note Nobody can use your personal account to download any application packages or process key requests. Please inform site administrator immediately if you login information are stolen or forgotten. You account would be locked if you let access to it to some one else.
  1. After your request would be processed and replied you will have to download your private key and store it to the directory you have installed the application.
