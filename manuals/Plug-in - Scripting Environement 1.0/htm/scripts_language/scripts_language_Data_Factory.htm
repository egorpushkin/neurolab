Scripting Environment 1.0 - Developer Guide
Data Factory Operations
Description
This package created to manage Data Factory objects. See Data Processing SDK documentation for the better understanding of data factory manipulations.
Package API
Creation/Destruction
createFactoryObject Creates new empty data factory object.
releaseFactoryObject Destroys existing data factory.
createFactory Initializes data factory object.
releaseFactory Releases factory data.
Factory Operations
setItemData Set new value for the specified data factory element.
fillFactory Fills data factory with the specified value.
increaseValue Increases value for the specified data factory element.
getItemData Returns current value of specified data factory element.
getDataLength Returns factory data length.
createFactoryObject
This function creates new data factory object. You can call releaseFactoryObject to release this object. All factories which has not been released would be destroyed by environment after script termination. 
Object createFactoryObject(
 [  Integer data_length
 [, Float fill_value = 0.0f ]]
);
createFactoryObject Parameters
data_length
[in] This argument specifies the length of new data factory object. If this argument absents environment will create empty data factory object.
fill_value
[in] This argument specifies the value to fill factory with.
createFactoryObject Return Value
Userdata object if successful; otherwise nil.
releaseFactoryObject 
Boolean releaseFactoryObject(
    Object data_factory
);
releaseFactoryObject Parameters
data_factory
[in] You must specify data factory object that you want to destroy. You can use this function for destroying only factory objects which were created using releaseFactoryObject function.
releaseFactoryObject Return Value
True if successful; otherwise false.
createFactory
This function allocates data buffer for the specified factory object. 
Void createFactory(
    Object data_factory,
    Integer data_length
 [, Float fill_value = 0.0f ]
);
createFactory Parameters
data_factory
[in] You must specify the factory object to operate with.
data_length
[in] This argument specifies the length of new data factory object.
fill_value
[in] This argument specifies the value to fill factory with.
releaseFactory
This function release data buffer for the specified factory object. 
Void releaseFactory(
    Object data_factory
);
releaseFactory Parameters
data_factory
[in] You must specify the factory object to operate with.
