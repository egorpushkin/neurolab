Scripting Environment 1.0 - Developer Guide
Elements Collection Operations
Description
The main idea of this package is to give an opportunity of enumerating network elements.
Package API
Enumerator Operations
createElementsEnumerator Initializes elements enumerator.
getNextElement Returns element at enumerator's position and moves it to the next one.
Collection Statistics
getInputsCount Returns number of signals required to be passed to network input.
getOutputsCount Returns number of signals that network passes to it's outputs.
createElementsEnumerator
This function sets enumerator position to the first network element. 
Void createElementsEnumerator(
    Integer enumerator_index
);
createElementsEnumerator Parameters
enumerator_index
[in] Specifies the number of enumerator that must be initialized. This parameter is extremely important. You can specify any number from range 6..9 otherwise function call has no effect. An opportunity of specifying enumerator number was added to the environment to make posible writing multithreaded plug-ins and trainers.
getNextElement 
Object getNextElement(
    Integer enumerator_index
);
getNextElement Parameters
enumerator_index
[in] Specifies the number of enumerator that must be used. You must specify here the same number that was passed during createElementsEnumerator call.
getNextElement Return Value
Userdata object if successful; otherwise nil.
