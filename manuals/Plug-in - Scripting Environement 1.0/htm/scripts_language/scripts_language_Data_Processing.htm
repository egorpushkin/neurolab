Scripting Environment 1.0 - Developer Guide
Data Processing Package
Description
This package provides functionality for processing data objects. These objects can be used by the script for training and processing network, for storage temporary script data and ect. Scripting environment also provides an opportunity to attach data files to the scripting solution so all attached data can be used by any script from this solution. This package can be divided in three logical parts:
  • Data Processor Operations
  • Data File Operations
  • Data Factory Operations
Each of these parts operates with corresponding environment object: Data Processor - CScriptSolution, Data File - CDataFile, Data Factory - CDataFactory. You can get more information about these objects using Data Processing SDK documentation.
