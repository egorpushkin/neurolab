Network Processor Operations
Description
The functiolnality of INetworkProcessor object has been implemented to power scripts achine with an opportunity of processing network with custom data and verify results of network processing. This opportunity is extremely important for scripts that realize network training algorithms.
Package API
Data Flow Operations
setInputData Passes data from specified data factory object to network inputs.
setOutputData Passes data from specified data factory object to network outputs.
getInputData Receives data from network inputs and collects it into the specified data factory.
getOutputData Receives data from network outputs and collects it into the specified data factory.
