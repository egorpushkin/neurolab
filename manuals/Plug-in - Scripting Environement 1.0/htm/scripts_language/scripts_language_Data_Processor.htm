Data Processor Operations
Description
Data Processor object was implemented by the enviromnent to give an opportunity of operating with data attached to the solution. You can add to the solution as many data files you want. Data Processor API allows you to enumerate these data for latest usage.
Note that you can add only files compatible with CDataFile data objects. See Data Processing SDK documentation for details.
Package API
This API provides all necessary functionality for enumerating attached data objects.
Operations
getDataFilesCount Returns count of attached data objects.
getDataFile Returns specified data object.
getDataFileName Returns file name for specified data object.
getDataFilesCount 
Integer getDataFilesCount();
getDataFilesCount Return Value
This function returns a number of valid attached data objects. Data objects is valid if CDataFile serializes it successfully.
getDataFile 
Object getDataFile(
    Integer data_object_index
);
Object getDataFile(
    String data_object_file_name
);
getDataFile Parameters
data_object_index
[in] This parameter specifies index of the data object to be returned.
data_object_file_name
[in] This parameter specifies file name of the data object to be returned. You must specify only name of a file without extension.
getDataFile Return Value
Userdata object if successful; otherwise nil.
getDataFileName 
String getDataFileName(
    Integer data_object_index
);
getDataFileName Parameters
data_object_index
[in] This parameter specifies index of the data object.
getDataFileName Return Value
String object that contains name of the data object with specified index if successful; otherwise nil.
