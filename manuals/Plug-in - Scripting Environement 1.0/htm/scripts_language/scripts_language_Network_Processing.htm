Network Processing Package
Description
This package provides functionality for processing prepared neural network. You have an opportunity to parse network; train elements' weights; specify network input data, process it and receive results from network outputs. This package can be divided in three logical parts:
  • Network Processor Operations
  • Elements Collection Operations
  • Network Element Operations
Each of these parts operates with corresponding environment object: Network Processor - INetworkProcessor, Elements Collection - IElementsCollection, Network Element - IElement, IDataFlow, IConnectionsFactory, IDataConnector, ILayerControl. You can get more information about these objects using Object Model documentation department.
