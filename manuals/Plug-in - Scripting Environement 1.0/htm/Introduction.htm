Scripting Environment 1.0 - Developer Guide
Introduction
Solution Description
Scripting Environment is a plug-in for the Neuro Laboratory Environment. It allows writing custom scripts for training, verification and testing built networks.
Scripting Environment Features
You can perform the following using Scripting Environment:
  • Create/Load solution with a various number of executable scripts.
  • Attach any data files you need to this solution.
  • Write script which has full access to the neural network built in the entironment and to all attached data files.
Your scripts would have access to control interfaces IElementsCollection and INetworkProcessor (through API provided by scripts interpreter). See Object Model - Network control interfaces department for more information.
